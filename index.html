SocketScout

A high-performance concurrent port scanner built in Python with an asyncio-driven pipeline, per-host isolation, and modular service interrogation. SocketScout reflects my interest in networking internals, systems correctness, and practical security tooling.

  • Scans multiple hosts and large port ranges concurrently using non-blocking I/O with explicit per-host state isolation.
  • Supports TCP connect scanning by default, optional SYN-based scanning, and modular banner grabbing for service metadata extraction.
  • Designed around clean separation between scan orchestration, networking primitives, validation, and output formatting.
